Basic next js react app initialisation containing page folder with completed react tutorial on next js documentation.


Make sure node js is installed before doing this

first run npm i
run npm run dev to run the development server locally hosted on 3000.


app folder contains components. page is the home page with the button component and layout defines structure for page components. Edit and change at will. The page contains use of 
react state, props and component exporting to get started. This is just a playground feel free to test anything.
